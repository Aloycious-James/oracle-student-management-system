# Student Management System (Oracle SQL & PL/SQL)

## Description
This project implements a Student Management System using Oracle SQL and PL/SQL.
It demonstrates real-world database design, stored procedures, functions,
triggers for validation and auditing, and role-based security.

## Technologies
- Oracle Database XE
- SQL
- PL/SQL
- SQL*Plus

## Features
- Student and department management
- Data validation using constraints and triggers
- Audit logging for insert and delete operations
- Stored procedures and functions
- Role-based access control

## How to Run
1. Connect to Oracle XE (XEPDB1)
2. Execute scripts in the following order:
   - database/*.sql
   - plsql/procedures/*.sql
   - plsql/functions/*.sql
   - plsql/triggers/*.sql
   - security/*.sql
